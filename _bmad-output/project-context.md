# Project Context for AI Agents

## Technology Stack
- Frontend: React + TypeScript + Vite
- UI: Tailwind CSS
- Backend: Node.js + Express
- Database: PostgreSQL
- Auth: Supabase Auth
- Testing: Playwright + Vitest
